Record potentially chargeable gifts (PETs and CLTs). Gifts become fully exempt from IHT after 7 years. Taper relief reduces the IHT charge for gifts made 3–7 years before death.

Backup

Export and restore all gifts, expenditure, settings and proof files.

Export backup

The backup zip contains everything needed to fully restore the app after a reinstall:

  • All gifting out of income records
  • All annual exemption records
  • All expenditure / income table data
  • All settings (gift maker details, recipients, income types, regular income/expenditure, font preferences)
  • All proof files (uploaded documents and images)

To restore after reinstall: install the app, start it, go to Backup → Import backup zip, and select your backup file. All data and files will be restored exactly as they were.
